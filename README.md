Drupal Gov 2017 - Drupal 8 and Microservices
============================================

## Anatomy of this project

* **api** - Demo API for central storage
* **site 1** - Demo Site 1
* **site 2** - Demo Site 2
* **proto** - Builds proto definitions for grpc
* **slides** - Slides for this presentation

## Usage

**Start environments**

```bash
make start
```

**Seed some demo articles**

```bash
make seed
```
